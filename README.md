## 📚 About WEBLEARN

WEBLEARN is a **frontend-based educational project** that helps students strengthen their web development skills using interactive learning.

✨ The platform combines **learning + quizzes + time challenges** to make practice more engaging.

### 💡 Features

- 📘 Structured learning modules  
- 🧠 Topic-wise quizzes  
- ⏱️ Timer-based quiz system  
- 🎯 Beginner friendly interface  
- 💻 Fully frontend implementation  

---
