# **App Name**: CodeClash Quizzes

## Core Features:

- Landing Page: Display logo, motivational quote, and login/signup options on a clean gradient background.
- Login/Signup: Allow users to create accounts and log in using email and password.
- Module Index: Present a categorized list of learning modules (HTML, CSS, JavaScript) with clickable topics.
- Submodule Pages: Display topic overviews with explanations, code snippets, output previews, and a 'Start Quiz' button.
- Timed Quizzes: Present time-based quizzes with multiple-choice questions, instant feedback, and explanations.
- Score Tracking: Keep track of user scores and times and store to Firestore database
- Leaderboard: Display user rankings based on score and time, calculated and displayed using data from Firestore

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke a sense of intellect and focus.
- Background color: Light purple (#EDE7F6), providing a gentle, distraction-free backdrop.
- Accent color: Soft lavender (#D1C4E9) for buttons and active elements, contrasting yet harmonizing with the primary.
- Body and headline font: 'Inter' sans-serif font for a modern and neutral look suitable for both headings and body text.
- Code font: 'Source Code Pro' for clear code snippet display.
- Simple, clear icons representing each module and topic.
- Subtle transitions between pages and quiz questions to maintain user engagement.
